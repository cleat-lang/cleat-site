# cleat-site

The marketing and documentation site for [Cleat](https://github.com/cleat-lang/cleat), a compiled orchestration language for auditable AI backends.

## Structure

Plain static HTML. No build step. No framework.

- `index.html` — landing
- `why.html` — comparisons
- `docs.html` — documentation
- `examples.html` — examples gallery
- `blog.html` — blog index
- `changelog.html` — release notes
- `sponsor.html` — sponsorship
- `404.html` — not found

## Deploying

Auto-deploys to Cloudflare Pages on every push to `main`.
